<#
	.SYNOPSIS 
 	#
	SYNOPSIS

	.DESCRIPTION 
 	#
	DESCRIPTION

	.PARAMETER name
	[string] - Name for the resulting thin clone.

	.PARAMETER snapshotName
	[string] - Name for the existing snapshot. This should follow the volumegroup:snapshot naming convention.

	.PARAMETER volumeGroupName
	[string] - Name of the volume group that contains the volume you wish to clone.

	.PARAMETER volumeName
	[string] - Name of the volume that you wish to clone.

	.EXAMPLE
	This creates a thin clone of `Volume01` named `ThinClone01` based on the existing snapshot named `Volumegroup01:Snapshot01`.
	

	.EXAMPLE
	New-SDPVolumeThinClone -name ThinClone01 -snapshotName 'Volumegroup01:Snapshot01' -volumeGroupName Volumegroup01 -volumeName Volume01
	

#>
